## ğŸ¨ Customization

### Updating Content
Edit `js/knowledgeBase.js` to update chat bot responses:

```javascript
const KnowledgeBase = {
    education: {
        keywords: ['education', 'study', 'university'],
        response: `Your updated education info here...`
    },
    // Add more categories...
};
```

### Changing Colors
Edit CSS variables in `css/style.css`:

```css
:root {
    --primary: #6366f1;      /* Main purple */
    --secondary: #ec4899;    /* Pink accent */
    --accent: #8b5cf6;       /* Purple accent */
    // ... more colors
}
```

### Adding Sections
1. Add HTML in `index.html`
2. Add styles in `css/style.css`
3. Update navigation in navbar
